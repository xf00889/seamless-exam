Usage Guide

Including Components

All components are located in templates/components/ and can be included using:

{% verbatim %}{% include 'components/component_name.html' with param1='value1' param2='value2' %}{% endverbatim %}

Available Components

  • navbar.html - Navigation bar with user info and logout
  • footer.html - Page footer with copyright
  • alert.html - Alert messages (success, error, warning, info)
  • messages.html - Django messages framework integration
  • pagination.html - Pagination controls for lists
  • form_field.html - Django form field renderer
  • input_field.html - Standalone input field renderer

Requirements Compliance

All components comply with:

  • Requirement 17.1 - All CSS loaded from external Tailwind files
  • Requirement 17.3 - No inline CSS styles in templates
  • Requirement 17.2 - All JavaScript in external files
  • Requirement 17.4 - No inline JavaScript in templates

For detailed documentation, see templates/components/README.md
